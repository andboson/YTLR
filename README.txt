YouTube Lite Reloaded plugin for Popcorn Hour

Features
--------
Watch youtube on your NMT

Prerequisites
-------------
Syabas powered NMT
Lighttpd web server (php5-curl required)
installed and activated NMT services (e.g. torrent).

Installation
------------

Installation now goes via NMT Community Software Installer - http://www.nmtinstaller.com
After installation add web-service by address:
http://<PCH-ADDRESS>:9999/YTLR_web/

Usage
-----
Use onscreen keyboard for search videos.



Keyboard Assignment
------------------
   DELETE  = deletes one simbol from search string
